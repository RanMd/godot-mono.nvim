# myplugin.nvim

A minimal Neovim plugin written in Lua.

## Installation

Use your favorite plugin manager, e.g. with lazy.nvim:

```lua
{
  'yourusername/myplugin.nvim',
  config = function()
    require('myplugin').hello()
  end
}
```
